# AI Prompt Maker & Gallery Browser

![AI Prompt Maker](https://raw.githubusercontent.com/wonseokjung/solopreneur-ai-agents/main/agents/kodari/assets/kodari_success.png)

## 🚀 프로젝트 개요
전문가급 AI 프롬프트를 쉽고 빠르게 생성하고, 프리미엄 AI 아트 갤러리를 통해 사용자 체류 시간을 극대화하는 **구글 애드센스 수익화 특화형 앱브라우저**입니다.

## ✨ 핵심 기능
- **AI Prompt 빌더**: 주제, 화풍, 조명을 조합하여 미드저니/DALL-E용 프롬프트 즉시 생성.
- **수익형 갤러리**: 고단가 키워드와 연계된 프리미엄 AI 이미지 피드.
- **인터랙티브 UI**: 사이버-프리미엄 테마와 실시간 알림 시스템.
- **No-Build 아키텍처**: 별도의 설치 없이 브라우저에서 바로 실행 가능.

## 🛠️ 기술 스택
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- **Kodari Persona Engine** (AI 서포트)

## 📦 설치 및 실행
1. 저장소를 클론합니다: `git clone <repository-url>`
2. `index.html` 파일을 브라우저로 엽니다.
